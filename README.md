# Prime Factorization Tree

Interactive Pluto notebook for visualizing prime factorization trees.

## Run in Browser
Click the badge below to launch the interactive notebook:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/john212/prime-factorization-tree/main?urlpath=pluto/open?path=factor_tree.jl)

## What it does
- Enter any number (1-10000)
- See the prime factorization tree
- View the exponential form of the factorization
